Coders of Delhi is a data-driven project focused on analyzing and processing user data to build social media–style recommendation features. The project involves data cleaning, preprocessing, and exploratory data analysis to extract meaningful insights from raw datasets.

Using Python, the system implements recommendation logic for features such as “People You May Know” and “Pages You Might Like” based on user interactions and relationships. The project demonstrates practical application of data analysis techniques and basic recommendation system concepts to improve user engagement.

This project highlights skills in data preprocessing, data analysis, and logic-based recommendations, making it a strong foundational project for data science and analytics roles.
